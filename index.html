COBOT

Spring 2022

User-friendly, accessible robot system for assistance in performing everyday tasks.

The arm assembly took place according to the instructions provided by Chris Annin's robot kit.

×

The ROS2 system is what controls the COBOT. It is hosted on a Linux laptop that is connected to the COBOT inside of the power box (shown below).

×

Speech Recognition

The COBOT utilizes speech recognition for detecting when a user would like to pick up a can using "" and signalling the ROS2 system to take the appropriate action.

×

EMG

The EMG subsystem will make use of an EMG sensor to partially control the COBOT system using one's own muscles.

×

Visual Recognition

The main goal of the Visual Recognition system is to have the COBOT be able to detect hand gestures (principally stop, thumbs up, and thumbs down) In addition, this subsystem will have the COBOT be able to detect objects visually so that it can pick them up.
