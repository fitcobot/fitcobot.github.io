ROS2

The ROS2 system is what controls the COBOT. It is hosted on a Linux laptop that is connected to the COBOT inside of the power box. Included below is a system diagram of how the ROS2 system works with the other interfaces.

×

Speech Recognition

The COBOT utilizes the CMUSphynx speech recognition engine for detecting when a user would like to perform a certain task and signalling the ROS2 system to take the appropriate action. Included below is a diagram for the different possible states.

×

EMG

The EMG subsystem will make use of an EMG sensor to partially control the COBOT system using one's own muscles.

×

Visual Recognition

The main goal of the Visual Recognition system is to have the COBOT be able to detect hand gestures (principally stop, thumbs up, and thumbs down) In addition, this subsystem will have the COBOT be able to detect objects visually so that it can pick them up.
